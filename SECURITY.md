# Security Policy

Report security issues privately by emailing `shorty@peyton.app`.

Please include:

- affected Shorty version or commit
- macOS version
- steps to reproduce
- expected and observed behavior
- whether Accessibility permission, the Safari extension, or the browser bridge
  is involved

Do not open a public issue for a vulnerability until there is a coordinated fix
or mitigation.

Shorty is distributed without warranty under AGPL-3.0-or-later. Security fixes
will be handled in the public repository once disclosure is safe.
