Local routing

One shortcut intent, app-specific output.

Built-in and reviewed adapters translate canonical shortcuts into native keys, menu commands, or pass-through behavior.

Native apps first

Bundled adapters load locally and work without a browser bridge or external service.

Browser bridge optional

Supported Chrome-family browsers can report selected web app domains when you install the bridge.

Diagnostics stay visible

The menu bar popover shows active coverage, permission state, bridge state, and adapter warnings.

Canonical keys

Your hands keep the same map.

Shorty keeps common intent names stable while the active app receives the shortcut it already understands.
