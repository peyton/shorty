macOS menu bar shortcuts

One shortcut set across your Mac.

Shorty runs in the menu bar and translates the shortcuts you already know for the app in front.

Download for macOS Setup help

Releases include shorty-<version>-macos.zip and a matching .sha256 checksum file.

Local-first from the first launch.

Download the archive, compare the SHA-256 checksum, then grant the one macOS permission Shorty needs for shortcut translation.

Download

Use the latest GitHub release asset named shorty-<version>-macos.zip.

Verify

Compare the archive with the matching .sha256 file before opening the app.

Allow

Grant Accessibility access so Shorty can listen for shortcuts and translate supported actions.

Checksum command shasum -a 256 shorty-<version>-macos.zip

Native first. Browser bridge optional.

Built-in adapters cover native Mac apps. The browser bridge is an explicit install for supported Chrome-family browsers.

  • Bundled native adapters load automatically.
  • Generated adapters require review before saving.
  • Browser domains are limited to supported web apps by default.

The same intent, app by app.

Shorty maps a small canonical shortcut set to the native command each app already exposes.

Command palette Shift Command P
Search everywhere Command K
Close current tab Command W

Support paths are short.

Start with the Accessibility and bridge checks. Email shorty@peyton.app with the macOS version, app name, and shortcut if the behavior still looks wrong.
