macOS menu bar shortcuts

One shortcut set across your Mac.

Shorty runs in the menu bar, watches the front app, and remaps a small trusted set of shortcuts locally.

Download for macOS Setup help

Releases include shorty-<version>-macos.zip and a matching .sha256 checksum file.

A direct download with a visible checksum.

Download the archive from the latest release, compare the SHA-256 line, then drag Shorty to Applications.

Download

Use the latest GitHub release asset named shorty-<version>-macos.zip.

Verify

Open the matching .sha256 file and compare it with the archive before opening the app.

Allow

Grant Accessibility access when macOS asks so Shorty can read menus and install its keyboard event tap.

Checksum command shasum -a 256 shorty-<version>-macos.zip

Native first. Browser bridge optional.

Shorty works without cloud services. Browser support is an explicit install for supported Chrome-family browsers.

  • Bundled native adapters load automatically.
  • Generated adapters require review before saving.
  • Browser domains are limited to supported web apps by default.
