Privacy

Shorty privacy

Shorty is a local-first shortcut tool. Questions can be sent to shorty@peyton.app.

Local processing

Shorty reads menu item names and keyboard shortcut metadata exposed by the active macOS app. Shortcut matching and event handling happen on the Mac.

Accessibility access

macOS Accessibility permission is used to inspect application menus and install the keyboard event tap. You can remove that access in System Settings at any time.

Browser bridge

The browser bridge is optional. By default, the extension reports only supported web-app domains such as Notion, Slack, Gmail, Google Docs, Figma, and Linear. All-domain reporting is an advanced diagnostic mode and is off by default.

Local storage

Shorty stores reviewed adapters and bridge socket state under the user's Application Support directory. It does not require a cloud account or external service.

Support email

Support messages sent to shorty@peyton.app are used to answer the request and diagnose the reported issue.

Contact shorty@peyton.app.
