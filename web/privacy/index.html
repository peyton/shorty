Privacy

Shorty privacy

Shorty is designed for local shortcut lookup. Questions can be sent to shorty@peyton.app.

Local processing

Shorty reads menu item names and keyboard shortcut metadata exposed by the active macOS app. The lookup work happens on the Mac.

Accessibility access

macOS accessibility permission is used to inspect application menus. You can remove that access in System Settings at any time.

Support email

Support messages sent to shorty@peyton.app are used to answer the request and diagnose the reported issue.

Contact shorty@peyton.app.
